# **App Name**: StudyFlow

## Core Features:

- Calendar View: Display a calendar view to show scheduled study sessions.
- Syllabus Input: Allow users to input their syllabus and import study tasks accordingly.
- AI Syllabus Input: Use AI tool to import the syllabus of the study session, to properly prepare you and make it more easier.
- Task Creation: Enable users to add new study tasks with details like subject, topic, duration, and notes.
- Upcoming Tasks: Show a list of upcoming scheduled tasks in order of time.
- Task Completion: Let users mark tasks as completed, archiving them or indicating progress.

## Style Guidelines:

- Background color: White (#FFFFFF) for a clean and distraction-free interface.
- Primary color: A soft blue (#A0C4FF) for buttons and key interactive elements, conveying a sense of calm and focus.
- Accent color: A deeper, slightly brighter shade of blue (#74A6F1) to highlight important actions and notifications.
- Body and headline font: 'Roboto' (sans-serif) for readability and a modern, clean aesthetic. Note: currently only Google Fonts are supported.
- Use simple, outlined icons in the primary blue color (#A0C4FF) to represent different study-related actions and categories.
- Maintain a clear and structured layout with generous spacing to prevent visual clutter and ensure easy navigation.
